## Steps to recreate Part 1

<ol>
    <li>Create project (group = com.username, artifactId = app name)</li>
    <li>Create RestController first with a list data structure to store jobs</li>
    <li>Set up basic HTTP Request Handling: Post and Get Requests</li>
    <li>Set up HTTP client like Postman or HTTPie</li>
    <li>Create service instead of using the data structure</li>
    <li>Implement counter for jobs</li>
    <li>Implement GetRequest for single job</li>
    <li>Implement ResponseEntity and HttpStatus</li>
    <li>Implement Delete Request</li>
    <li>implement Put Request</li>
    <li>Use RequestMapping at the class level</li>
</ol>
